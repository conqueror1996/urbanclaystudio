# UrbanClay Studio

Premium material inspiration engine implementation.

## Features
- **Onboarding Flow**: Multi-step wizard for identity and project details.
- **Taste Calibration**: Tinder-style swipe interface (`SwipeView`).
- **Design System**: Premium "Apple-level clean" UI with Tailwind v4, Inter, and Cormorant Garamond.
- **Mock Generation**: Simulated backend analysis and board curation.

## Tech Stack
- Next.js 16 (App Router)
- Tailwind CSS v4
- Framer Motion (Animations)
- Lucide React (Icons)
- TypeScript

## Getting Started

```bash
npm install
npm run dev
```
